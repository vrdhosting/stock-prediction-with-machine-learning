# stock-prediction-with-machine-learning
A full-stack stock prediction portal built with a Django REST backend and a React + Vite frontend. The Project is containerized using Docker Containerization and PostgreSQL is used as Database.
______________________________________________________________________________________________________________
# Overview
This project provides a machine learning-backed stock prediction application with user registration and API-driven frontend integration. The backend is built with Django and Django REST Framework, while the frontend uses React and Vite.
